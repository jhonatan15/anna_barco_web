    Proyecto página web Anna-Barco (Zapatería)

    Pagina web estatica para una zapateria, aun esta en proceso pero basicamente es para mostrar los estilos y galeria de los zapatos
    
    Como clonarlo o descargarlo:
    Puedes descargar el contenido y abrir el archivo Index.html dentro de la carpeta.
    
    Como ejecutarlo
    Para ejecutarlo solo debes abrir el archivo index.html
